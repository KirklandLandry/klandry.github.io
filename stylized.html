Stylized Rendering

Using various techniques to achieve a non-photorealistic rendering style.

Cel Shader

A cel/toon shader I created featuring:

  • Support for all light types
  • Shadows
  • Environmental reflections
  • Rim lighting on unshadowed pixels
  • Metallic, occlusion, height, and roughness maps (baked into 1 rgba texture via custom texture baking tool)
  • Normal maps
  • Emission maps
  • Transparency
  • Secondary detail/normal maps

Outline Effect

The outline effect I created starts by assigning vertex colours to assign where edges will appear (edges will appear along different vertex colour edges). In Unity, a post process replacement shader renders all the vertex colours to a texture. A vertex colour alpha of zero means there will be no vertex colour, so you can have both outlined shapes and non outlined shapes in a scene. The final shader then uses the camera normal and depth texture along with the vertex colour map to find all valid edges in the scene. The colour of the outline can be modified to be any solid colour or blend so that the outline is just a darker colour of the scene colour it's overlaying. The outline thickness can be changed as well.
